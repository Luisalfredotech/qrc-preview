# QRC Group — Non-Production Preview

This repository hosts a **static, non-production preview** of a redesign exploration for the QRC Group website (`qrcgroup.net`).

## Status

- **Non-production.** Nothing here runs in WordPress, on the live `qrcgroup.net` site, or against any production system.
- **For QRC client review only.** Intended audience is QRC Group stakeholders evaluating the proposed multi-page direction.
- **Static HTML only.** No backend, no database, no forms wired to a server, no analytics pixels.
- **Source vault remains private.** This repo contains only the published static preview files. The full design history, audit notes, and project documentation live in a separate private vault and are not exposed here.

## What's published

A 20-page static site under the root of this repo:

- `index.html` — homepage
- 7 main pages — `about`, `core-services`, `technology-services`, `proven-results`, `training`, `open-opportunities`, `contact`
- 8 Core Services detail pages
- 4 Technology Services detail pages
- `shared.css`, `shared.js` — shared layout + i18n
- `assets/` — QRC-supplied imagery (logo, credentials, hero, audience photos)

## Language toggle

The preview supports English (default) and Spanish via `?lang=en` / `?lang=es`. Spanish is an AI-drafted preview pass and is awaiting native review.
